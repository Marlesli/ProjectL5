Concepts We've Learned

  1. DOM (Document Object Model)
    1. Parents whose links show directly back to children
    2. Your're able to manipulate the structure and its content
  2. Semantic vs. Non-Sematic
    1. Semantic elements are more clarified to the browser and non-semantic doesn't give clear information about the content
    2. Ex.<div> is non-semantic and <p> is semantic
  3. Block Elements vs. Inline
    1. Each block elements start on their own line and automatically there is margin around the element
    2. Inline elements do not start on a new line and take up only the space it needs to.
